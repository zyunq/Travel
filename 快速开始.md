# 快速开始 - Python座位表生成工具

## 🚀 3步开始使用

### 第1步：安装Python环境

**如果您已经有Python**，跳到第2步。

**如果没有Python**：

1. **下载Python**
   - 访问：https://www.python.org/downloads/
   - 点击 "Download Python 3.x.x"

2. **安装**（重要！）
   - 运行下载的安装程序
   - ✅ **勾选 "Add Python to PATH"**
   - 点击 "Install Now"

3. **验证**
   ```bash
   python --version
   ```

### 第2步：运行生成脚本

**最简单的方式**：

双击运行 `生成座位表.bat`

脚本会自动：
- ✅ 检查Python环境
- ✅ 安装依赖包（如果需要）
- ✅ 选择输入文件
- ✅ 生成座位表

### 第3步：查看结果

打开生成的Excel文件，您会看到：

**工作表1：乘客车票信息**
- 完整的乘客数据
- 表头样式美化
- 冻结首行

## ✨ 核心优势

相比Node.js版本，Python版本提供：

1. ✅ **专业3+2座位布局** - 符合真实二等座布局
2. ✅ **双工作表设计** - 信息+座位可视化
3. ✅ **数据自动清洗** - 过滤无效座位号
4. ✅ **日期格式转换** - 20260608 → 2026/06/08
5. ✅ **证件号保护** - 防止变科学计数法

## 🎯 使用场景

### 场景1：快速生成单个座位表
```bash
生成座位表.bat
```

### 场景2：批量处理多个文件
```bash
python tools\generate_seat_excel.py 文件1.xlsx 输出1.xlsx
python tools\generate_seat_excel.py 文件2.xlsx 输出2.xlsx
python tools\generate_seat_excel.py 文件3.xlsx 输出3.xlsx
```

### 场景3：集成到工作流
在您的批处理脚本中调用：
```batch
python tools\generate_seat_excel.py "%INPUT%" "%OUTPUT%"
```

## ❓ 常见问题

**Q: Python脚本在哪里？**
A: `tools\generate_seat_excel.py`

**Q: 输出文件在哪里？**
A: 默认在当前目录，文件名格式：`座位表_YYYYMMDD_HHMM.xlsx`

**Q: 支持哪些Excel格式？**
A: 只支持 `.xlsx` 格式（不支持 `.xls`）

**Q: 座位号格式要求？**
A: 标准格式如 `12A`、`8B`（数字+字母A-F）
