## 💡 使用技巧

### 技巧1：批量处理

创建批处理脚本：
```batch
@echo off
for %%f in (input\*.xlsx) do (
    python tools\generate_seat_excel.py "%%f" "output\%%~nf_座位表.xlsx"
)
```

### 技巧2：自定义文件名

```bash
python tools\generate_seat_excel.py 输入.xlsx "自定义名称.xlsx"
```

### 技巧3：处理多个车厢

Python脚本自动：
- 识别所有车厢
- 按车厢号排序
- 分组显示座位

## 🔧 故障排查

### 问题1：找不到Python

**症状**：`'python' 不是内部或外部命令`

**解决**：
1. 确认Python已安装
2. 重新安装时勾选"Add Python to PATH"
3. 或使用完整路径：`C:\Python39\python.exe`

### 问题2：openpyxl未安装

**症状**：`ModuleNotFoundError: No module named 'openpyxl'`

**解决**：
```bash
pip install openpyxl
```

### 问题3：Excel文件格式错误

**症状**：`未读取到有效乘客数据`

**检查**：
- Excel必须是 `.xlsx` 格式
- 必须包含必需列：姓名、车厢、座号
- 座位号格式：数字+字母（如：12A）
